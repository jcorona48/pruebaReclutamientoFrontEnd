# Instructions

## Requirements
[BunJs]:https://bun.sh/
[NodeJs]:https://nodejs.org/en
**[BunJs]** - JavaScript runtime that can execute TypeScript.

**[NodeJs]** - JavaScript runtime.

## Steps

To run this program, you can use both runtimes. Here are the commands:

### NodeJs

~~~
node revertArray.js
~~~

### BunJs

~~~
bun revertArray.ts

//OR

bun revertArray.js
~~~
