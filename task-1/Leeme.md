# Instrucciones

## Requisitos
[BunJs]:https://bun.sh/
[NodeJs]:https://nodejs.org/en
**[BunJs]**  - Runtime de JavaScript que puede ejecutar TypeScript.

**[NodeJs]** - Runtime de JavaScript.


## Pasos

Para Ejecutar este programa lo puede hacer con ambos runtime aqui los comandos:

### NodeJs

~~~
node revertArray.js
~~~

### BunJs

~~~
bun revertArray.ts

//OR

bun revertArray.js
~~~
